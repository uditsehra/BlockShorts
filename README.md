## 🚀 How It Works

The extension:

1. Injects scripts into YouTube pages
2. Detects Shorts-related elements
3. Removes or hides them dynamically
4. Prevents navigation to Shorts
5. Uses MutationObserver to handle infinite scroll

---

## 🛠️ Installation (Local / Personal Use)

### Option 1: Run via Xcode

1. Open project in Xcode  
2. Click ▶️ Run  
3. Open Safari → Settings → Extensions  
4. Enable the extension  

---

### Option 2: Install `.app`

1. Build & export:
   ```
   Product → Archive → Distribute App → Copy App
   ```
2. Share `.app` file  
3. Open app → Enable in Safari  

---

## 🧪 Usage

1. Go to YouTube  
2. Enable extension  
3. Enjoy distraction-free browsing  

---

## ⚙️ Toggle Control

Use the popup:

- ✅ ON → Blocks Shorts  
- ❌ OFF → Restores Shorts
